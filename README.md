# Curso completo de Python
Este repositorio contiene muy buenos ejercicios y prácticas para empezar a programar en Python, organizados en diferentes carpetas para mantener un aprendizaje ordenado y progresivo. 
## Contenido 
- - Variables
- - Tipos de datos (int, float, str, bool)
- - Entrada y salida de datos (input() y print())
- - Operadores matemáticos
- - Operadores de comparación
- - Operadores lógicos
- - Condicionales (if, elif, else)
- - Bucles for
- - Bucles while
- - Listas
- - Acceso a elementos de listas
- - Métodos de listas
- - Funciones
- - Parámetros y argumentos
- - Valor de retorno (return)
- - Importación de módulos (import)
- - Módulo random
- - Clases
- - Objetos
- - Constructor __init__
- - Método __str__
- - Atributos de instancia
- - Encapsulación básica
- - Programación Orientada a Objetos (POO)
- - Uso de varias clases en un programa
- - Simulaciones sencillas

## Objetivo: aprender los fundamentos de Python mediante ejercicios y ejemplos prácticos, mejorando poco a poco estas habilidades de programación.

## Tecnologías utilizadas: - Python 3
